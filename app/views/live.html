NHS login user journeys

Some of these journeys will use the same screens but with different parameters depending on what is happening. For information on these parameters, go to how to use.

  1. Update 2024 / NHS Wales App service

    User journey - NHS App Wales login uplift

    NHS App after doing POL using provisioning to uplift their NHS login account to highest level.

    1. Existing account uplift email
    2. Existing account recovery email (edge case)
    3. Succes uplift

    Discovery 2023 / NHS App main service

    GP checks

    Provisioning will get it's own error codes to send back or be shared with GP staff.

    User journey - GP prescribing NHS App

    Allowing users to claim an NHS login account provisioned or uplifted by a GP.

    1. New account set up email
    2. Existing account uplift email
    3. Existing account recovery email
    4. Succes new account
    5. Succes uplift (same as new account)
    Errors

    The same checks are done to see if a user has changed state since the GP checks (journey 1), these screens are shown to the user.

    • Already has submitted a PYI application
    • Already uplifted (same as link expired)
    • Account disabled
    • Account disabled - User dead with account
    • Account disabled - User dead without account
    • Duplicate email (P5)
    • Duplicate email (P9)
    • Duplicate NHS number/ multiple P5 account exists
    • Duplicate NHS number/ one P5 account exists
    • Email link lockout
    • Generic error
    • Link expired (same as already uplifted)
    • OTP expired
    • OTP lockout
    • PDS data doesn't match
    • PDS mobile no longer matches
    • P9 account already exists
    • Timeout
    • Wrong code entered
    • Wrong password
